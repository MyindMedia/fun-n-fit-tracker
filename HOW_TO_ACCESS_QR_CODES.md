# HOW TO ACCESS STUDENT QR CODES

## Step-by-Step Guide

### Method 1: Via Search Bar (EASIEST)

1. **Look at the top navigation bar** - You'll see a search box that says "Find Athlete..."

2. **Type the student's name** in the search box

3. **Click on the student** from the dropdown results

4. **Student profile modal opens** - You'll see 4 tabs at the top:
   - Trophy Case
   - Rewards Shop
   - Coach Pep Talk
   - **📱 QR Code** ← Click this tab!

5. **The QR Code displays** with options to:
   - 🖨️ Print QR Card (creates a printable card)
   - 💾 Download QR Code (saves as PNG file)

---

### Method 2: Via Admin Dashboard

1. Go to **Admin** tab in top navigation
2. Click **ATHLETES** tab
3. Click on any student's card
4. This opens their profile
5. Click the **📱 QR Code** tab

---

## What the QR Code Contains

Each QR code is UNIQUE to the student and contains:
- Student ID
- Student Full Name
- Type: CHECKIN

When scanned with the QR Scanner, it automatically marks that specific student as present.

---

## How to Use QR Codes for Check-In

### Setup (One Time):
1. Open each student's profile
2. Click **📱 QR Code** tab
3. Click **🖨️ Print QR Card**
4. Print the card and give it to the student (or laminate it)

### Daily Check-In:
1. In Admin Dashboard, click **📱 QR Check-In** button (top right)
2. Camera opens
3. Students scan their QR cards
4. System automatically marks them present!

---

## Troubleshooting

**"I don't see the QR Code tab"**
- Make sure you're opening a student profile (search for their name first)
- The tab is labeled "📱 QR Code" and is the 4th tab

**"QR Code shows 'Generating QR Code...'"**
- Wait 1-2 seconds, it generates automatically
- If stuck, close and reopen the profile

**"QR Scanner doesn't detect the code"**
- Make sure the printed QR code is clear and not blurry
- Hold the code steady in front of the camera
- Ensure good lighting

**Check Browser Console for Errors:**
- Press F12 to open developer tools
- Look for messages starting with 🔄, ✅, or ❌
- These show QR code generation status

---

## Student ID Format

Each student has a unique ID like: `a1b2c3d4-e5f6-7890-abcd-ef1234567890`

The QR code stores this ID so the scanner knows exactly which student to check in.
